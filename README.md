# MalinaJS Development Server
Just simple http server with reload and autoroute capabilities.

### Folder structure ###
```
malina-project/
├ src/
│ ├ cmp/
│ │ └ [your component files]
│ ├ module/
│ │ └ [your module files]
│ ├ store/
│ │ └ [your store + param files]
│ ├ lib/
│ │ └ [your lib files]
│ ├ pages/
│ │ └ [your page routes]
│ ├ App.xht
│ ├ main.js
│ └ routes.js[autoroute]
├ public/
│ └ [your publish folder]
├ package.json
├ options.js
└ malina.config.js 
```
