# MalinaJS Development Server
Just simple http server with hot, reload and autoroute capabilities.

### Folder structure ###
```
malina-project/
├ src/
│ ├ components/
│ │ └ [your component files]
│ ├ modules/
│ │ └ [your module files]
│ ├ stores/
│ │ └ [your store + params files]
│ ├ libs/
│ │ └ [your lib files]
│ ├ pages/
│ │ └ [your page routes]
│ ├ App.xht
│ ├ main.js
│ └ routes.js[autoroute]
├ public/
│ └ [your publish folder]
├ package.json
├ options.js
└ malina.config.js 
```
